# tour-plan

Tour Plan
Учебный проект на курсе Web-start в Glo Academy

## О проекте

Верстка главной страницы сайта турфирмы

## Функционал

- Адаптивная, валидная, кросбраузерная верстка
- Слайдеры
- Валидация форм
- Отправка формы на email
- Модальное окно
- Parallax эффект
- Анимация

## Стек технологий

- HTML
- CSS
- SASS
- JQuery
- Плагины JQuery
- PHP Mailer
- Pixel perfect
- BEM
